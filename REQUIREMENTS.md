## Functional Requirements

| ID | Title | Description | Status |
|----|-------|-------------|--------|
| REQ-1 | BR-1: Weekly meal suggestions | Generate 7-day meal plan based on user's ingredients, budget, and preferences. One click, calendar view. | ✅ Implemented |
| REQ-2 | BR-2: Missing ingredients list | Auto-create categorized shopping list with ONLY ingredients the user does NOT have. | ✅ Implemented |
| REQ-3 | BR-3: Mode selection | Smart Mode (batch cooking, leftovers) or Fresh Mode (variety). Affects scoring algorithm. | ✅ Implemented |
| REQ-4 | BR-4: User Authentication & Data Privacy | Register, login, logout. Each user's data is private. JWT tokens. | ✅ Implemented |
| REQ-5 | BR-5: Ingredient-based meal suggestions | Add fresh or leftover ingredients. Smart Mode prioritizes leftovers (+3). Fresh Mode penalizes leftovers (-2). | ✅ Implemented |
